Mercado

Confira!

Esboço de um sistema de mercado feito utilizando HTML5 e CSS3, que pode ser ultilizado para fazer diversos tipos de sistemas diferentes, esse é focado em venda de produtos por exemplo, é possível ver os itens a venda, quantidade disponível e o valor de cada item numa tabela, além disso você ainda pode adicionar os itens em um carrinho.
